# discounboot  
회원가입/로그인을 하고 상품을 조회하며 장바구니에 추가/삭제하는 웹 애플리케이션입니다.  

# 개발환경 설정
윈도우 10 사용  
* 자바 버전 확인  
  ```  
  > javac -version  
  javac 1.8.0_231  
  ```
* 이클립스 2019-09 버전 설치  
https://www.eclipse.org/downloads/  
* 스프링부트 STS 설치  
Eclipse - Help - Eclipse Marketplace - Spring Tool 3 Add-On for Spring Tools 4 3.9.12.CI  
* MySQL 5.7 버전  
* Visual Studio Code  
* Vue-CLI 2.5.16 버전  
  ```  
  $ npm install vue  
  ```

# 특징  
* 회원가입 시 이메일 중복 확인, 가입 연령 제한, 비밀번호 확인  
* 로그인    
* 로그인 시 토큰 생성  
* 배너 한 페이지에 5개씩 조회  
* 상품 목록 한 페이지에 6개씩 조회  
* 로그인 시 장바구니 기능  
* 상품 장바구니에 추가/ 삭제  
* 상품 상세보기 시 글/ 사진 형태로 노출
